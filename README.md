# TinyURL Service

TinyURL is a service where users can create short links, such as tinyurl.com/3rp36a3s, redirecting to longer links, such as https://www.adroit-tt.com.

## Running

### Requirements
- node.js
- npm
- dotnet

###

### Run client

```bash
cd client
npm install
npm run dev
```

### Run api

```bash
cd api
dotnet run
```
